Project Details

This project has been build for learning purposes using Bubble.io
The application contains three pages:

  • Home page: which lists the available offers where the candidate can filter the offers according to the following criteria: by category (Developer, Analyst, product manager ...), by the type of contact (CDD , CDI, Freelace).
  • Offer Details page: which includes the detailed description of the position and the possibility of applying for this offer.
  • Employer Space page: which lists the offers posted and the candidates attached to it, the employer has the possibility of adding and editing an existing offer.
